##Instruction	
####To compile
* You need to  :
* include headers files in "zeldaLib/include"
* include lib files in "zeldaLib/lib"
* If you don't have installed requiered dll, you can find it in "zeldaLib/dll"

####To play	
* 'up' 	to move left
* 'right' 	to move right
* 'up' 	to move up
* 'down' 	to move down
* 'b' 	weapon change
* 'space' 	attack
* 'm'		show minimap
* 'ESC'	quit
